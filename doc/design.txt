-------------------------------------
Смена дизайна для игры
-------------------------------------

Чтобы изменить дизайн игры, можно
1. отредактировать стиль CSS в корневом style.css
2. указать html-файл, который будет шаблоном вывода локации

-------------------------------------
HTML-файл - шаблон игры
-------------------------------------

Допускаются любые теги, классы. Id, кроме специальных, также можно использовать.

--- Заголовок, текст, выбор:

Специальные id - определяют место для части игры
их можно применять в ЛЮБОМ ПАРНОМ теге
id="title"  - в этом теге будет выводиться заголовок локации
id="text"  - описание локации
id="choice"  - текст выбора. Этот тег и именно этот, с id="choice"
        будет повторяться для каждого выбора (сколько их есть в локации)

id="backlink"  - код для маркера "Вернуться", то есть для отметки выборов,
                 которые ведут на предыдущую локацию

Эти id НЕ останутся в html игры - они исчезнут.

Для дизайна через CSS также можно использовать сочетание class и сочетание тегов

---  Параметры:

Параметры выводятся в тегах с аттрибутом  data-param
Какой именно параметр - указывает этот аттрибут. Писать надо так же, как в исходнике игры
пример: <b data-param="player.health"></b>

По умолчанию параметры выводятся также, внутри парного тега
Вывод в виде аттрибута представляется спорным, наверное,
лучше уж в будущем придумать js-биндинг и полноценные мини-скрипты
